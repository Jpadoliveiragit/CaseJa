# Case Já - Sistema de Organização de Casamentos

Projeto desenvolvido para a Etapa 2 do trabalho avaliativo, com foco na organização de informações importantes para casamentos.

## Integrantes

- João Pedro Almeida
- Felipe Gabriel

## Descrição

O Case Já é um sistema para ajudar casais e organizadores a controlar fornecedores, tarefas e gastos relacionados ao casamento em um único lugar.

Neste segundo commit, o projeto possui a estrutura inicial, tela inicial aprimorada, layout base melhorado, navegação simples e uma tela de fornecedores organizada para receber o CRUD nos próximos commits.

## Tecnologias Utilizadas

- React
- JavaScript
- Vite
- HTML
- CSS

## Como Executar o Projeto

```bash
npm install
npm run dev
```

Depois, acesse o endereço exibido no terminal pelo Vite.

## Funcionalidades Previstas

- Cadastro de fornecedores
- Listagem de fornecedores
- Edição de fornecedores
- Exclusão de fornecedores
- Controle de tarefas do casamento
- Controle de gastos
- Armazenamento dos dados no LocalStorage
